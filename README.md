ER diagram ensures clear database design.

## Setup Instructions
1. Install **XAMPP** and start Apache & MySQL.
2. Copy the project folder into `htdocs`.
3. Import `edoc.sql` into **phpMyAdmin** to create the database.
4. Access the system via `http://localhost/hospital_management_system`.
